### Movement
#### Keyboard
- Up Arrow to accelerate
- Down Arrow to brake
- Left/Right Arrow to steer
- x to transform

#### PS4 Controller
- R2 to accelerate
- L2 to brake
- Left stick to steer
- X to transform
